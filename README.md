# AI Learning — Sina

## How to Resume a Session
1. Read PROGRESS.md to see where you left off
2. Read the relevant topic file in topics/ if one exists
3. Tell Claude: "Resume my AI learning. I'm on Module N."

## File Structure
```
learning/
  README.md          — this file, orientation
  LEARNING_PLAN.md   — full 12-module curriculum with concepts and exercises
  PROGRESS.md        — session log, status per module, mastered concepts
  topics/            — notes per module, created as we go
    01-llm-foundations/
    02-prompt-engineering/
    ... etc
  exercises/         — code and experiments from hands-on work
    module-01/
    module-02/
    ... etc
```

## How Sessions Work
- Each session we pick up where we left off or dive into a specific module
- I explain concepts, you ask questions, we build things
- Notes get saved to topics/
- Code gets saved to exercises/
- Progress.md gets updated at the end of each session
